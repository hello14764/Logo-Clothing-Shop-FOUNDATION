# Website images

Photos for the **live site only**. Not for sales PDFs (those live in `content/sales/`).

## Folders

| Folder | Put here |
|--------|----------|
| `hero/` | Homepage hero, page headers |
| `case-studies/` | One subfolder per slug, e.g. `weldaloy-client-gifting/` |
| `general/` | Logo on site, team, shop, production shots |

## Before adding files

1. Compress at [squoosh.app](https://squoosh.app) — aim under 200 KB per image
2. Use JPG for photos, PNG for logos with transparency
3. Descriptive filenames: `weldaloy-charger-box.jpg` not `IMG_4523.jpg`

## Deploy note

Everything in `site/` can go live. This folder is part of the public website.
