# Brand colors — Logo Clothing Shop (Astro site)

Used in `tailwind.config.mjs`. Logo Clothing Shop name until launch; palette previews Stitch & Stone direction.

## Core palette

| Token | Hex | Tailwind | Use |
|-------|-----|----------|-----|
| Navy | `#0A2540` | `navy` | Primary buttons, links, footer, headings |
| Navy dark | `#061829` | `navy-dark` | Hover states, dark bands |
| Stone | `#D2C8B6` | `stone` | Cards, borders, subtle surfaces |
| Gold | `#C9A66B` | `gold` | Eyebrows, thread dividers, tags |
| Warm gray | `#F5F2ED` | `warm-gray` | Page background |
| White | `#FFFFFF` | `white` | Content cards, inverse button text |

## Rules

- **Primary CTA:** navy
- **Gold:** accents only (not primary buttons)
- **Footer:** navy background
- **No em dashes** in customer-facing copy (Phase 1 pages; case study JSON copy pass optional)

## Legacy static site

The former `site/` folder is deprecated. Live build output is `dist/` from `npm run build`.
