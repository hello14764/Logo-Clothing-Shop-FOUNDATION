# Brand assets

Master files for Logo Clothing Shop (becoming Stitch & Stone Co.).

## Logo

| File | Use |
|------|-----|
| `logo-current/logo-clothing-shop-tall.png` | **Master copy** — website, print, sales |

## Copy rule for one-pager PDF

The sales one-pager (`content/sales/one-pager.html`) needs the logo **in the same folder** to print correctly.

If you update the logo here, copy it:

```
brand/logo-current/logo-clothing-shop-tall.png
  → content/sales/logo-clothing-shop-tall.png
```

When the website launches with Stitch & Stone branding, add new files under `logo-stitch-stone/` (create when ready). Do not delete Logo Clothing Shop assets until redirect is live.

## Website vs sales

- **Website** will eventually use logos from `site/images/` or `brand/` via build copy
- **Sales PDF** uses the copy in `content/sales/` only
